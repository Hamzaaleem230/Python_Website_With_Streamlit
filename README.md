## 🚀 Getting Started
1. Clone the repository

```bash
git clone https://github.com/hamzaaleem230/portfolio-app.git
cd portfolio-app
```

---

2. Install dependencies
```bash
pip install -r requirements.txt
```

---

3. Run the app

```bash
streamlit run app.py
```
